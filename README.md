# SILVAS CORE - Portfolio Website

Portfolio de Armando J. Silvas - Designer

## 🚀 View Live

Go to: https://github.com/ajsrsiete-afk/SILVAS-CORE

## 📁 Files

- index.html - Main portfolio page
- styles.css - CSS styling

## 🎨 Features

- Responsive design
- Smooth animations
- Color scheme: Navy & Light Blue
- Fonts: Impact & Kodashon

## 📝 To Customize

1. Update your name and text
2. Add your images
3. Update social media links
4. Add your fonts to fonts/ folder

## 🚀 Deploy with GitHub Pages

1. Go to Settings → Pages
2. Select main branch
3. Your site will be live!
